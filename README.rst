3D objects
==========

Compilation
-----------

Libraries
 * OpenGl
 * Free GLUT

To compile program::

    gcc src/* -lGL -lGLU -lglut -lm -o U4Aurelijus

User interface
--------------

To view object, drag with mouse.
[1] to turn on/off axis
[2] to turn on/off game mode
[Q][W], [A][S], [Z][X] to rotate around red, gree and blue axes.
